Grammar Worksheet — Interrogative Sentences (Yes / No)
Active  : Had + Subject + V3 + Object + ?
→
Passive : Had + Object + been + V3 + by + Subject + ?
Example — Active: Had the boy caught the butterfly?  →  Passive: Had the butterfly been caught by the boy?
